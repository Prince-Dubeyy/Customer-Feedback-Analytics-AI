# 📌 Overview

ProductPulse AI is an AI-powered SaaS platform designed to help organizations analyze customer feedback at scale.

The platform automatically processes customer reviews, complaints, suggestions, and feature requests, transforming them into actionable business insights using Artificial Intelligence, Natural Language Processing (NLP), Business Analytics, Historical Memory, and Runtime Intelligence.

Built as a hackathon project, ProductPulse AI demonstrates the integration of:

- Data Science
- NLP
- Business Intelligence
- AI Agents
- Memory Systems
- LLM Routing
- Full Stack Development
- Cloud Deployment

---

# 🎯 Problem Statement

Businesses receive thousands of customer feedback entries every day.

Traditional analysis methods struggle to:

- Detect recurring complaints
- Identify feature requests
- Track sentiment trends
- Monitor changes over time
- Generate executive-level insights
- Prioritize customer concerns

ProductPulse AI automates this entire workflow.

# ✨ Core Innovations

## 🧠 Hindsight Engine

The Hindsight Engine provides persistent memory across feedback uploads.

Instead of analyzing datasets independently, the system remembers historical customer feedback and performs trend comparisons.

### Example

Upload 1:

```text
Checkout Complaints = 10
```

Upload 2:

```text
Checkout Complaints = 20
```

Generated Insight:

```text
Checkout complaints increased by 100%
```

### Benefits

- Historical trend detection
- Temporal awareness
- Emerging issue identification
- Customer experience monitoring

---

## ⚡ cascadeflow Runtime Intelligence

cascadeflow intelligently routes AI requests based on:

- Complexity
- Cost
- Latency
- Budget constraints

The system maintains a complete audit trail for every AI interaction.

### Runtime Metrics

- Model Used
- Request Cost
- Latency
- Routing Reason
- Fallback Status

### Benefits

- Reduced AI costs
- Transparent AI operations
- Budget-aware inference
- Improved reliability

# 🚀 Getting Started

## Backend Setup

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Linux / Mac
source venv/bin/activate

pip install -r requirements.txt

python main.py
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

# ☁️ Deployment

Root Directory

```text
backend
```

Build Command

```bash
pip install -r requirements.txt
```

Start Command

```bash
uvicorn main:app --host 0.0.0.0 --port $PORT
```
